BehatMailExtension.

This extension was written out of my fascination with Behat as an automatic test framework that clicks through your web application like a normal user. We had done some extensive testing at work using this technology but testing the e-mails was kind of impossible. On a search for a solution I quickly realized that no feasible solution existed so I wrote one. This extension enables the developer to test E-Mails send to an inbox by the application.

The Anti Social Network. Speak Your Mind.

The anti social network was a project initially started as hackathon project at my current workplace. I had the idea of a social media platform with features working not as intended e.g. login form error would present the username of the entered password. Lately that idea changed to a more real anti social approach by giving the user only the choice to dislike content. Everybody has to be connected with everyone and unfriending someone can not be undone creating some awkward situations.

Tech Stack:
  • PHP 7.1
  • Symfony 4
  • Bootstrap 4
  • Twig
  • jQuery and JavaScript
  • Fontawesome
